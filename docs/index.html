MonGone - MongoDB Resource Optimization Tool

Streamline your MongoDB Atlas environment by automatically scaling down unused resources and converting fixed resources to auto-scaling configurations. Reduce costs and improve database efficiency.

View on GitHub Documentation

Effortless Installation

Install MonGone quickly using pipx:

pipx install git+https://github.com/raestrada/MonGone.git@v0.3.0

Zero to Optimal

Automatically identify unused MongoDB clusters and databases, scale them to zero, or remove them entirely. Convert static resources to auto-scaling to save on costs and improve cloud efficiency.

Generate and Execute Plans

Generate an optimization plan and let MonGone execute it for you. Track which resources are costing you money and take control of your database infrastructure.

Cost Analysis

MonGone adds cost data to each resource to help you make informed decisions and better manage your cloud database expenses.
